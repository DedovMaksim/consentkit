# ConsentKit

Modern cookie consent and privacy preferences plugin for WordPress.

ConsentKit helps website owners manage cookie consent, privacy preferences and conditional script loading with a clean user experience and WordPress-native architecture.

## Features

- Modern cookie consent banner
- Preferences modal with granular consent
- Cookie categories:
  - Required
  - Analytics
  - Functional
  - Marketing
- Conditional script loading
- Google Consent Mode v2 support
- Floating preferences button
- Accessibility support
- Reduced motion support
- Translation-ready architecture
- Multisite support
- Lightweight class-based architecture

## Installation

1. Upload the plugin files to the `/wp-content/plugins/consentkit` directory
2. Activate the plugin through the WordPress admin panel
3. Open **ConsentKit → Settings**
4. Configure your banner, preferences and scripts

## Usage

### Cookie Banner

ConsentKit displays a configurable cookie banner for visitors who have not yet saved their preferences.

### Preferences Modal

Visitors can:
- accept all cookies
- reject optional cookies
- configure categories individually

### Script Management

Optional scripts are loaded only after consent is granted.

Supported categories:
- Analytics
- Functional
- Marketing

### Google Consent Mode v2

ConsentKit can update Google Consent Mode v2 preferences automatically based on visitor choices.

## Accessibility

ConsentKit includes:
- keyboard navigation
